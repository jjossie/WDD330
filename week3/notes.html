Notes

  • Object literals are a powerful way of passing parameters into functions. They allow you to use unordered named parameters in JS functions as well as specify default values for them, which can be really handy. An example: 
    
function triathalon({run, bike, swim}){
    if (run)
        console.log(`I ran ${run} miles`);
    if (bike)
        console.log(`I biked ${bike} miles`);
    if (swim)
        console.log(`I swam ${swim} miles`);
}
triathalon({run: 4, bike: 30})
triathalon({swim: 2, bike: 25})


Console:
> I ran 4 miles
> I biked 30 miles
> I biked 25 miles
> I swam 2 miles
