Week 9 Notes

Chapter 14

  • HTML5 introduced important features like the data- attribute and APIs for things like localStorage, geolocation, multimedia, and web workers.

    • The data- attribute

  • An HTML tag can have attributes that start with "data-" which can be used to store data within an HTML element purely for Javascript.
  • Ignored by the browser
  • A data attribute "data-key" could be accesesed either using element.dataset.key (notice how "data-" is omitted) or element.getAttribute("data-key")
  • Hyphens would be replaced with camel case, so "data-max-length" becomes "maxLength" when accessed through element.dataset.maxLength

    • Storage

  • LocalStorage lasts between refreshes and opening/closing the browser while SessionStorage resets between sessions.
  • LocalStorage items can be accessed either using localStorage.setItem("key", "value") or localStorage.key = "value"

    • HTML APIs

  • geolocation can be accessed using the navigator.geolocation object
  • geolocation.getCurrentPosition(callback) gives a location position object to a callback passed in
  • Position has properties like speed, altitude, heading, and most importantly, coordinates. It also even has the accuracy of said coordinates.
  • it also has a method called watchPosition() that takes a callback that will run everytime the position is updated.
